Testando Media Queries

Media queries

São uma técnica do CSS3 usada para aplicar estilos diferentes a um documento, dependendo das características do dispositivo em que ele está sendo exibido, como a largura da tela,a altura,a resolução,e orientação(retrato ou paisagem),entre outras. Eles são essenciais para criar designs responsivos que se ajustam a diferentes tamanhos de tela, garantindo uma experiência de usuário consistente em dispositivos móveis,tablets e desktops.

Exemplo Básico

Aqui está um exemplo básico de como usar media queries para ajustar o layout de uma página com base na largura da tela:
/* Estilos padrão para todos os dispositivos */
body { background-color: lightgrey; font-size: 16px; }
/* Estilos para dispositivos com largura de tela de até 600px */ @media only screen and (max-width: 600px) { body { background-color: lightblue; font-size: 12px; } }
/* Estilos para dispositivos com largura de tela entre 600px e 1200px */ @media only screen and (min-width: 601px) and (max-width: 1200px) { body { background-color: lightgreen; font-size: 14px; } }
/* Estilos para dispositivos com largura de tela superior a 1200px */ @media only screen and (min-width: 1201px) { body { background-color: lightcoral; font-size: 18px; } }

Como Funcionam

@media only screen and (max-width: 600px): Aplica estilos se a largura da tela for de até 600 pixels.

@media only screen and (min-width: 601px) and (max-width: 1200px): Aplica estilos se a largura da tela for entre 601 e 1200 pixels.

@media only screen and (min-width: 1201px): Aplica estilos se a largura da tela for superior a 1200 pixels.

Utilidades de Media Queries

Responsividade:Ajustar o layout para diferentes tamanhos de tela, como smartphones, tablets, e desktops. Acessibilidade:Melhorar a usabilidade e experiência do usuário em dispositivos com diferentes capacidades e recursos. Design Adaptativo: Aplicar estilos específicos para dispositivos com características únicas, como alta resolução ou modos de tela (paisagem/retrato). Media queries são uma ferramenta poderosa para garantir que seu site seja acessível e funcional em uma ampla gama de dispositivos.
